🌐 Interactive Navigation Menu

An interactive and responsive navigation menu built with HTML, CSS, and JavaScript.
The menu features smooth hover effects, color/style changes on scroll, and remains fixed at the top for easy access across all pages.

✨ Features

🔝 Fixed Position – Always visible while scrolling.

🎨 Dynamic Styling – Changes color/style when scrolling.

🖱️ Hover Effects – Highlight menu items on hover.

📱 Responsive Design – Works seamlessly on desktop and mobile.

⚡ Lightweight & Fast – Minimal code with smooth transitions.

🚀 Tech Stack

HTML5 – Structure

CSS3 – Styling & transitions

JavaScript (ES6) – Scroll and hover interactivity
